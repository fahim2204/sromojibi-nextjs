# Sromojibi - Bangladesh Worker Directory

Sromojibi is a web application built with Next.js 14 (App Router), Tailwind CSS, and Hero UI (NextUI) to connect homeowners and businesses with trusted local workers, mistris, and trade technicians across Bangladesh.

## ✨ Features

- **SEO-Optimized Server Components**: Fast rendering, SSR structured data, canonical URLs, and Open Graph tags.
- **Worker Category Directory**: Electrician, Plumber, Rajmistri, Tiles Worker, Painter, Carpenter, AC Technician, CCTV Installer.
- **Location Based Coverage**: Division & district directory pages for Dhaka, Mymensingh, Chittagong, Sylhet, Rajshahi, and Khulna.
- **Worker Registration**: Free profile submission form for mistris and trade specialists.
- **Knowledge Base & Guides**: Practical home maintenance and worker hiring guides.

## 🚀 Technology Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Hero UI (NextUI)
- **Icons**: Emoji & SVG assets

## 📦 Installation & Commands

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
